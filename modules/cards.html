Cards

Responsive image + heading + text card grid — model overviews, feature grids.

A responsive grid of image + heading + text cards — the classic "feature grid" or "model overview" layout.

How it works

Each row you add becomes one card. Whichever cell contains only an image becomes the card's image; the other cell becomes its body text. The block automatically arranges all cards into a responsive grid.

When authors use it

Use for model overviews, feature highlights, or any "pick one of these" grid of items.

What you fill in

Card imageOne image per card.
Card bodyHeading + short text per card.
