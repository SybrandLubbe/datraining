Specification Table

Categorized technical data table (engine, chassis, brakes...).

A categorized technical data table — the classic "full spec sheet" (engine, dimensions, brakes, etc.), grouped under category headings.

How it works

The first row is the table title. After that, whenever the first cell of a row has text, it starts a new category group; rows after it (with an empty first cell) are that category's label/value pairs.

When authors use it

Use on a model page's full technical-specifications section.

What you fill in

Title rowHeading for the whole table.
Category cellStarts a new group, e.g. "Engine".
Label / Value cellsOne spec per row, e.g. "Power" / "197 hp".
