Teaser

Compact image + heading + copy teaser, for secondary callouts.

Very similar to Cards — an image + heading + text item — but styled more compactly, with a smaller heading, for teaser-style callouts rather than a full feature grid.

How it works

Same authoring pattern as Cards: one image cell + one text cell per row. The block gives every heading a consistent, slightly smaller style than Cards uses.

When authors use it

Use for secondary callouts (e.g. "related content") where Cards would feel too heavy.

What you fill in

Teaser imageOne image per item.
Teaser bodyHeading + short text per item.
