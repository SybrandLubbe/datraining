Vehicle Selector

Tabbed model picker: click a model name, see its image + description.

A tabbed model picker: a row of clickable model names, each showing its own image and description.

How it works

Each row becomes one tab. The heading in that row becomes the clickable tab label; the rest of the row (image, description) is what displays when that tab is active. The block builds the tab navigation and image/description panels itself.

When authors use it

Use on a "Models" overview page where visitors compare several motorcycles.

What you fill in

Tab headingBecomes the clickable tab label (removed from the visible body text).
Tab imageShown when that tab is active.
Tab descriptionShown alongside the image when that tab is active.
