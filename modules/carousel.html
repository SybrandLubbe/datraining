Carousel

Horizontally swipeable slide carousel.

A horizontally swipeable set of slides (touch/swipe on mobile, click-through on desktop).

How it works

Each row is one slide. The block handles swipe gestures and the sliding transition itself — you only provide the slide content.

When authors use it

Use where visitors should browse several items side-by-side, like a set of accessories or a photo set from an event.

What you fill in

Slide contentOne row per slide — usually an image plus a short caption.
