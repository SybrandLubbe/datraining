Table

Plain HTML data table, no grouping or stat styling.

A plain, simple HTML data table — no categories, no special stat styling, just rows and columns.

How it works

Every row/cell you enter becomes a table row/cell, rendered as an ordinary table.

When authors use it

Use for simple tabular content that doesn't need Specification Table's grouping (e.g. a size/fit chart).

What you fill in

Rows/cellsWhatever you type, one table cell per authoring cell.
