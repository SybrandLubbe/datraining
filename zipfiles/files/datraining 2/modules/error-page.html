Error Page

The 404 / page-not-found content.

The content shown on a 404 / page-not-found state.

How it works

Mostly plain content, with one automatic behavior: any "home" link on the page is automatically pointed at the correct site root, so it still works correctly across every locale folder.

When authors use it

One per site (or per locale, if translated); not something you add to regular pages.

What you fill in

Message + home linkWhatever explanatory text you want, plus a link back home.
