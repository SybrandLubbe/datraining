Image

3 variants — single, two-image, and full-width sticky — each with a strict size/ratio checklist.

The Image block can be used in sections of a webpage where you'd like to showcase pictures — on a homepage or any content page like a 'Heritage' page. It has 3 variants.

The 3 variants

  • 2 images — the second image is smaller than the first one
  • Single image — image ratio is 16:9
  • Full width image — stretches edge to edge

1. Single image variant — UX checklist

Table: Image header row, then one row with <<add image here>>.

UX/UI checklist for this variant: Image ratio 16:9, 1440 × 810px (minimum).

2. Two images variant — UX checklist

Table: Image header row, then one row with two cells, each <<add image here>>.

UX/UI checklist: First image ratio 4:3, 1440 × 1080px (min). Second image ratio 4:3, 1440 × 1080px (min).

Images are displayed next to each other, except on Viewport XS, where they stack below each other.

3. Full-width sticky image variant — UX checklist

Table: Image (sticky) header row, then one row with <<add image here>>.

UX/UI checklist: Image ratio 16:9, 1440 × 810px (min).

The image stays sticky in the background while the rest of the page content scrolls over it.

Adding the block

Two ways to add it: copy the component block from an existing page under your respective country folder, or use the Library section from the da.live toolbar and choose the relevant variant from the list.

Bonus: a dark section background

Adding a Section Metadata block with Style: dark makes that section's background black. Add this at the end of a block (e.g. after an Image block) to make the background of that section dark.

Section Metadata
Style | dark

What you fill in

Image (single)One image. Ratio 16:9, minimum 1440 × 810px.
Image (two images)Two images side by side; the second is displayed smaller than the first. First image ratio 4:3, min 1440 × 1080px. Second image ratio 4:3, min 1440 × 1080px. They stack below each other only on viewport XS.
Image (sticky, full-width)One full-width image. Ratio 16:9, min 1440 × 810px. Stays sticky in the background while page content scrolls over it.
Section Metadata: Style = darkOptional companion block — add it after any block to switch that section's background to dark.

Single image variant rendered on the page — 16:9 ratio.

Two-image variant rendered side by side, the second smaller than the first.
