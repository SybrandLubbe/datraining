Dealer Locator

The full-width Woosmap-powered dealer map used at the top of the Dealer Page template.

The Dealer Locator block is the full-width interactive map at the top of the Dealer Page – Tabs template — a fully dynamic, Woosmap API-driven way of rendering dealer information, replacing the old setup where authors had to manually update columns and shift cells every time a dealer was added.

Why this template replaced the old manual setup

With the new Dealer Page – Tabs template:

  • Dealer data is automatically populated from Woosmap APIs
  • No manual column updates are required on the page when a dealer is added, removed, or changed
  • Navigation is simplified with tabs and accordions, even across many countries and dealers
  • The page layout adapts automatically as more markets are added

How the template is put together

Starting a new dealer page from the Dealer Page – Tabs template (as opposed to the older plain Dealer Page template) gives you three blocks stacked on the page, in this order:

  1. Metadata block — sets the page Title (default “Dealers”) and the global wooskey (your site's Woosmap API key).
  2. Dealer Locator block — the full-width, large-height interactive map, using that same global Woosmap key.
  3. Woosmap Dealers block (twice — once as “local dealers”, once as “global dealers”) — see the dedicated Woosmap Dealers module for how those two variants work.

What this means for you going forward

You no longer need to manually update dealer lists, shift table columns, or maintain “other country” sections by hand — all dealer information is now fully dynamic and sourced directly from Woosmap. In practice:

  • Any dealer update made in Woosmap automatically reflects on the site — there's nothing to republish on the page itself.
  • The page always shows local dealers first, based on the country the site is running in.
  • Navigation remains clean and scalable, no matter how many markets or dealers are added in the future.

What you fill in

Metadata: TitlePage title shown to visitors. Default: “Dealers”.
Metadata: wooskeyThe global Woosmap API key for this site — the same key used by both the Dealer Locator and the Woosmap Dealers blocks below it on the page.
Dealer Locator sizeFullwidth, Height: large — the configuration used in the standard template.
