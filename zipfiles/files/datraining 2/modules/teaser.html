Teaser

Card container with 2 background styles × 4 width ratios, unlimited cards.

The teaser component is a container for cards. It has 2 variations with respect to the background image (default, with-image) and 4 variations for size ratios (default, one-two, two-one, one-one).

Adding the block

Two ways to add it: copy the component block from an existing page under your respective country folder, or use the Library section from the da.live tool and choose Teaser from the list available.

1. Teaser variant default

Block name: Teaser. Cards show the image above the text, on the default (light) background. Add as many image+text rows as you like — they arrange into a responsive row of cards.

2. Teaser variant image

Block name: Teaser (image). Instead of the image sitting above the text, the card's image becomes the full background of the card, with the text content displayed on top of the image.

With-image ratio variants (3–5)

  • Teaser (image, one-two) — 30%–60% width split
  • Teaser (image, two-one) — 60%–30% width split
  • Teaser (image, one-one) — 50%–50% width split

All three keep the image-as-background behavior from variant 2, just at different width proportions.

Default-background ratio variants (6)

The same width ratios are also available with the default (non-overlaid) background:

  • Teaser (one-one) — 50%–50%
  • Teaser (two-one) — 60%–30%
  • Teaser (one-two) — 30%–60%

What you fill in

Card imageOne image per card.
Card titleBold heading, e.g. "Merchandising".
Card descriptionSupporting text under the title.
Price line (optional)e.g. "59,99 Euro".
Read More linkRendered as a styled button with an arrow, e.g. "Read More :arrow-right:".
